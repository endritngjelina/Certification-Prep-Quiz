# Certification Prep Quiz 🎓

A collection of interactive quiz games designed to help students prepare for popular certification exams. Each quiz is built using simple, self‑contained HTML, CSS, and JavaScript—no frameworks, no dependencies, just open and play.

## 📚 Included Quizzes
- **Design for Delight (D4D) Quiz Game**
- **Entrepreneurship & Small Business (ESB) Quiz Game**
- **WordPress Fundamentals Quiz Game**

## 🚀 Features
- 100% browser‑based — no installation required  
- Clean, mobile‑friendly UI  
- Instant feedback on answers  
- Easy to customize or extend with new questions  
- MIT‑licensed for personal or educational use  

## 🛠 How to Use
1. Clone or download the repository  
2. Open any `.html` file in your browser  
3. Start practicing  

## 📁 File Structure

Design_for_Delight_Quiz_Game.html 
Entrepreneurship_and_Small_Business_Quiz_Game.html 
WordPress_Quiz_Game.html


## 🧩 Want to Add Your Own Questions?
Each quiz is written in plain HTML/JS, so customizing questions is easy.
